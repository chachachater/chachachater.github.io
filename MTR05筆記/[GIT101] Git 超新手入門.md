# 什麼是版本控制？
> 報告用的 PPT 到底甚麼時候會是最終版

檔案的更改變動是一個連續的過程，**每次更改檔案並產生一個新的檔案版本** => 這個過程不斷 loop 之後會累積非常多版本，造成不好管理之類的問題，所以需要透過版本控制來解決這個困擾。
